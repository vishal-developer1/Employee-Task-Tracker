# Employee Task Tracker
This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 16.2.8.

## 🛠 Features

- Add tasks with:
  - Employee Name
  - Task Title
  - Priority (Low, Medium, High)
  - Due Date
- Display all tasks in a styled table
